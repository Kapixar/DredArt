    How to use Legacy Holo:

    1. Place these blocks on given coordinates (hold / aka ?)
      • Hatch - 24, 25
      • Sign - 24, 25
      • Sign(hover) - 24, 25
      • Sign(near) - 24, 25
    2. Drop the Zip into the game window
    3. Paint
    4. In case of invisible sprite:
      • Move close to the block
      • Max zoom in
      • Refresh the assets: Settings -> Modify assets -> Refresh

    How to use BlueHolo

    1. Drop blueHolo.zip into the game window
    2. Paste the blueprint into a BP Scanner
    3. Place blueprint in desired place
