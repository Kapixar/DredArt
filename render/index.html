About

This tool is part of DredArt extension made by I am Shrek.

It let's you create fine pixelarts from ordinary images.
Those pixelarts come in 2 forms:

  • color map
  • image to use in DredArt Extension

Render is extremly simple, fast and intuitive. Works with most of image files.

Do not use any obscene pictures. dredark.io developer is not affiliated with this project. Source code is available on GitHub on MIT license.
